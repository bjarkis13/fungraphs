Data Acqusition and Visualisation

This site was created as part of a project in the course "Data Acqusition and Visualisation" at Reykjavík University. Django 1.8.7 was used as a framework, skeleton was the basis for the stylesheet, jquery for the interactive map, based on this svg modified by hand and with inkscape for missing municipalities nvd3 for line charts and d3 for graphs.
